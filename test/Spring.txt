# Spring Framework
- 스프링 프레임워크는 자바 플랫폼을 위한 오픈 소스 애플리케이션 프레임워크로서 스프링(spring) 이라고도 합니다

# Spring Framework 특징
1. 제어 반전 ( IoC, Inversion of Control )
-  스프링에서 설정한 내용을 기반으로 애플리케이션 필요 요소들을 호출

2. 의존성 주입( DI, Dependency Injection )
- 상호 연결이 필요한 객체들을 스프링 설정에서 연결, 주입하도록 설정
- 상호 의존성을 낮춰 확장 및 교체가 용이한 구조로 만들 수 있다

3. 관전 지향 프로그래밍 ( AOP, Aspect Oriented Programming )
- 트랜잭션, 로깅, 보안 등 여러 모듈에서 공통적으로 지원하는 기능을 분리하여 사용할 수 있습니다

4. POJO ( Plain Old Java Object )
- 자바 객체의 라이프사이클을 스프링 컨테이너가 직접 관리하며, 스프링 컨테이너로 부터 필요한 객체들을 얻어올 수 있습니다
